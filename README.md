# Simulador de Factura de Ventas - Ecuador

## Descripción
Este proyecto corresponde al **Caso 1 del Taller 02** del curso de *Programación Web*, cuyo propósito fue desarrollar un **simulador de facturación** para una tienda ecuatoriana. El sistema permite registrar productos, calcular subtotales, aplicar el IVA del 15% y mostrar el total final a pagar, utilizando vectores paralelos y estructuras de control.

---

## Objetivos
- Implementar un sistema de facturación funcional utilizando vectores en JavaScript.  
- Aplicar operaciones aritméticas y condiciones compuestas para el cálculo de totales.  
- Simular un proceso comercial real mediante una interfaz web sencilla y profesional.  
- Integrar los conceptos de HTML, CSS y JavaScript en un proyecto práctico.

---

## Desarrollo
El sistema utiliza dos vectores principales:
- `productos[]`: contiene los códigos, precios y descripciones de los artículos disponibles.  
- `facturaDetalles[]`: almacena los productos seleccionados por el usuario junto con su cantidad y total.

La lógica está implementada con ciclos `for` para recorrer los vectores y condicionales `if` para aplicar las reglas de cálculo. Además, se incorpora un diseño visual tipo factura con estructura organizada y estilo profesional.

---

## Tecnologías Utilizadas
- **HTML5** – estructura del documento  
- **CSS3** – diseño y formato visual de la factura  
- **JavaScript (ES6)** – lógica funcional y manipulación de vectores  

---

## Resultados
El sistema genera dinámicamente una factura con todos los valores calculados (subtotal, IVA y total), permitiendo agregar y eliminar productos en tiempo real.  
Cumple con los requerimientos del caso planteado y demuestra el uso práctico de vectores en entornos web.
